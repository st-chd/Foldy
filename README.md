# Foldy

Foldy는 프롬프트, 로어북(월드 인포), 정규식 목록에 폴더 기능을 추가하는 확장 프로그램입니다. 항목이 많아졌을 때 관련 항목을 폴더로 묶어 접고 펼치거나, 폴더 단위로 이동·관리할 수 있게 해줍니다.

## 주요 기능 요약
- 폴더 생성, 일괄 이동, 전체 접기/펼치기, 내보내기/불러오기
- 폴더 이름 변경, 개별 항목을 다른 폴더로 이동
- 드래그 앤 드롭으로 폴더와 항목 순서 변경
- 폴더 색상, 테두리 색상, 이름 색상 설정 및 설정 일괄 반영
- (로어북) 폴더 안의 항목 전략/위치 일괄 변경
- (정규식) 폴더 안의 항목 일괄 토글(on, off)
- 사용중/미사용/전체 폴더 초기화

## 지원 영역

#### 프롬프트

프롬프트 목록에 폴더를 추가합니다. 프롬프트 순서를 폴더 기준으로 정리하며, 폴더 구조는 프리셋별로 따로 저장됩니다.

#### 로어북

로어북 편집기에서 정렬 방식을 `폴더 순서`로 바꾸면 폴더 UI가 표시됩니다.
로어북 항목을 폴더에 넣고, 폴더 단위로 전략·삽입 위치 설정을 일괄 변경할 수 있습니다.

#### 정규식

글로벌, 프리셋, 범위 정규식 목록에 각각 폴더를 추가합니다.
정규식 스크립트가 많을 때 용도별로 묶어 관리할 수 있습니다.
폴더 단위로 설정을 일괄 토글할 수 있습니다.


## 내보내기와 불러오기

현재 두 가지 방식의 내보내기를 지원하며, 프롬프트/로어북/정규식 각 영역에서 동일하게 동작합니다.

| 방식 | 설명 |
| --- | --- |
| 폴더 구조만 | 현재 항목 내용은 그대로 두고 폴더 배치만 백업합니다.|
| 내용 포함 번들 | 프롬프트/로어북/정규식 내용과 폴더 구조를 함께 백업합니다. |

다른 환경으로 옮기거나 큰 폴더 정리를 하기 전에는 내용 포함 번들을 먼저 내보내 두는 것을 권장합니다.
불러오기 시 기존 내용을 덮어쓰는 경우, 진행 전에 확인 메시지가 표시됩니다.

## 설정 패널

설정 패널에서는 다음 작업을 할 수 있습니다.

- 프롬프트/로어북/정규식 폴더 사용 여부 변경
- 사용중인 폴더, 미사용 폴더, 전체 폴더 중 삭제 범위 선택
- 프롬프트/로어북/정규식/전체 폴더 초기화

초기화는 <b>폴더만 삭제합니다.</b> 원본 프롬프트, 로어북 항목, 정규식 스크립트 내용은 삭제하지 않습니다.

## 참고 사항

- Foldy 폴더는 한 단계만 지원합니다. 폴더 안에 하위 폴더를 만들 수는 없습니다.
- 드래그 앤 드롭이 잘 되지 않는 환경에서는 항목의 `폴더로 이동` 또는 `일괄 이동` 버튼을 사용할 수 있습니다.
- 미사용 폴더는 현재 존재하지 않는 프롬프트, 로어북, 정규식에 연결된 폴더 기록입니다.
